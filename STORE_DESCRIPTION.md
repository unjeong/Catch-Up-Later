I knew it

## 짧은 설명 (132자 이내)
특정 웹사이트의 새 게시글을 자동으로 감지하고 뱃지로 알려주는 확장 프로그램. SSO 로그인 사이트도 지원!

## 상세 설명

🔔 **I knew it** - 새 글을 놓치지 마세요!

특정 웹사이트에 새로운 게시글이 올라오면 자동으로 감지하여 알려드립니다!

### ✨ 주요 기능

🔍 **스마트 자동 감지**
- CSS 선택자 자동 감지
- 한 번의 클릭으로 게시판 패턴 인식
- 다양한 게시판 형태 지원

🔔 **Gmail 스타일 뱃지 알림**
- 새 글 개수가 아이콘에 표시
- 한눈에 새 글 확인
- 브라우저 알림 지원

🔐 **SSO/로그인 사이트 지원**
- 브라우저 세션 활용
- 회사 인트라넷, 사내 게시판 모니터링 가능

⏰ **유연한 설정**
- 1분~1시간 체크 주기 선택
- 여러 사이트 동시 모니터링

### 🚀 사용 방법

1. 모니터링할 사이트에 로그인
2. 확장 프로그램 아이콘 클릭
3. "자동" 버튼으로 게시글 선택자 감지
4. "등록" 클릭
5. 완료! 새 글이 올라오면 뱃지로 알림

### 📌 이런 분들께 추천

- 회사 공지사항을 놓치고 싶지 않은 분
- 특정 커뮤니티의 새 글을 빠르게 확인하고 싶은 분
- 채용 공고, 이벤트 소식을 실시간으로 받고 싶은 분

### 🔒 개인정보 보호

- 모든 데이터는 사용자 브라우저에만 저장
- 외부 서버로 데이터 전송 없음
- 가벼운 용량으로 빠른 동작
