# 🔔 I knew it

특정 웹사이트에 새 게시글이 올라오면 바로 알려주는 크롬 확장 프로그램입니다.

## ✨ 주요 기능

- 🔍 **자동 감지**: CSS 선택자 자동 감지로 간편하게 설정
- 🔔 **뱃지 알림**: 새 글 개수를 아이콘에 표시 (Gmail 스타일)
- 🔐 **SSO 지원**: 로그인된 사이트도 모니터링 가능
- ⏰ **주기 설정**: 1분~1시간 간격으로 설정

## 🚀 설치 방법

### 1. 확장 프로그램 로드

1. 크롬에서 `chrome://extensions` 접속
2. **개발자 모드** 활성화
3. **압축해제된 확장 프로그램을 로드합니다** 클릭
4. `new-post-alert-extension` 폴더 선택

### 2. 아이콘 생성 (필요시)

1. `generate-icons.html` 파일을 브라우저에서 열기
2. **모든 아이콘 다운로드** 클릭
3. 다운로드된 파일을 `icons` 폴더에 저장

## 📖 사용 방법

1. 모니터링할 사이트에 **로그인**
2. 게시판 목록 페이지로 이동
3. 확장 프로그램 아이콘 클릭
4. **🔍 자동** 버튼으로 선택자 감지
5. **+ 등록** 클릭
